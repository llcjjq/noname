# 天赐拓展

> 无名杀（Noname）扩展包 —— 朝阳·天赐

## 📖 简介

天赐拓展为无名杀添加了一名新武将 **朝阳·天赐**（群雄·冯势力），拥有4个技能，定位为卡牌玩家型角色。

## 🧑‍🤝‍🧑 武将信息

| 项目 | 内容 |
|:----|:------|
| 武将名 | 天赐 |
| 称号 | 朝阳 |
| 势力 | 冯（feng） |
| 体力 | 4 |
| 技能 | 逆天运气、监管者鸡龙、冯冯杀、辨忠奸 |
| 定位 | 卡牌玩家 |

## 🎴 技能说明

### 1️⃣ 逆天运气 — 回合开始
> *"运气也是实力的一部分！"*

- 弃置 1~2 张牌
- **弃1张**：从牌堆获得一张指定的**基本牌**
- **弃2张**：从牌堆获得一张指定的**基本牌** + 一张指定的**锦囊牌**
- 牌堆中每没有一张你指定的牌，你**摸2张牌**

### 2️⃣ 监管者鸡龙 — 常驻
> *"嗷呜~！"*

- 每使用或打出一张牌，获得一个 **"嗷！"标记**
- **回合开始**：判定。若点数 > 标记数 → 弃2牌或受1伤；若 < 标记数 → 摸2牌或回1血
- **回合结束**：同上

### 3️⃣ 冯冯杀 — 每轮调整
> *"冯冯杀！调整形态！"*

- 每轮可选择修改【监管者鸡龙】：
  - 🔄 **反转比较**（大于↔小于）
  - ❌ **移除回合开始**判定
  - ❌ **移除回合结束**判定

### 4️⃣ 辨忠奸 — 主公技（游戏开始）
> *"忠奸善恶，我自明辨！"*

选择一名玩家，令其获得以下技能之一：

#### 🛡️ 护忠
- 天赐对你的伤害 → **防止**，双方各**摸2张**
- 本回合首次受伤后，后续伤害全部**免疫**

#### ⚔️ 诛奸
- 天赐对你的伤害 **+1**
- 天赐每次对你造成伤害 → **摸2张**
- 你死亡时：
  - 忠臣 → 天赐**减1体力上限**
  - 反贼 → 天赐**摸3张 + 加1体力上限**

## 📦 安装方法

1. 将 `天赐拓展` 文件夹复制到无名杀目录下的 `noname\resources\app\extension\` 中
2. 打开无名杀，进入 **扩展管理**
3. 启用 **天赐拓展**
4. 创建房间时选择武将 **天赐** 即可使用

## 🔧 图片 & 音频自定义

### 头像
- 默认头像：`tianci.jpg`
- 皮肤：`tianci2.jpg`（可在选将时切换）
- 文件格式：JPG，建议尺寸 200×200px

### 语音
- 命名规则：`技能名编号.mp3`（如 `逆天运气1.mp3`）
- 支持格式：MP3

| 语音文件 | 触发时机 |
|:---------|:---------|
| `逆天运气1.mp3` | 回合开始发动逆天运气 |
| `监管者鸡龙_phaseBegin1.mp3` | 回合开始判定 |
| `监管者鸡龙_phaseEnd1.mp3` | 回合结束判定 |
| `冯冯杀1.mp3` | 每轮开始调整 |
| `辨忠奸1.mp3` | 游戏开始辨忠奸 |
| `die1.mp3` | 阵亡 |
